English:

1) Execute the file Simulator_de_modos_propios.exe
2) If a Windows Smart Screen warning message appears when starting the program, choose "More information"> "Run anyway"
3) If it is detected by Windows Defender, add the program to the allowed list
4) If this is the first time the application has been run on the computer, the COMSOL Multiphysics® runtime environment necessary to run the application will be downloaded first.

*********************

Español:

1) Ejecutar el archivo Simulador_de_modos_propios.exe
2) Si al iniciar el programa aparece un mensaje de advertencia de Windows Smart Screen, elegir "Más información" > "Ejecutar de todas formas"
3) Si es detectado por Windows Defender, agregar el programa a la lista de permitidos
4) Si es la primera vez que se ejecuta la aplicación en el ordenador, se descargará en primera instancia el runtime environment de COMSOL Multiphysics® necesario para correr la aplicación.
